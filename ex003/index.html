Teste de links

Primeiro teste de links, como o do youtube usamos o código <a href="link para redireiconameto"</a>

lembrando que para o link abrir em outra pag basta você digitar "target=_blank"

Teste de marcaçao

Primeiro teste com o marcador onde usamos a tag <mark>

agora com o sublinhado que usamos o <u>

ja o deletado usamos o <del>

para negríto usamo a tag <strong> e para italico a tag <em> que vem de "enfase"

Midias

Para usar as midias, como imagens,videos e musicas usaremos as tags "img", "source", entre outras opçôes

    <source media="(min-width: )" srcset="" type="image/">:
    <source src="" type="">
etc...

Listas

Listas com o <ol> e o <ul> definindo o tipo das "ol" com "type"(podendo usar o "a", "A", "1" e "I" para numeros romanos. ) e sempre usando o <li>:
EX:

  1. item 1
  2. item 2
  3. item tres

Exemplos de midias

imagem <img src="" alt="">
favicon se usa a tag <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">
